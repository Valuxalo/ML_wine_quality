# Датасет:

Датасет классификации качества вина. https://www.kaggle.com/datasets/yasserh/wine-quality-dataset. Содержит 11 признаков и имеет 1143 объекта. 
Целевой переменной является оценка качества вина [5, 6, 7].

# Модели:

Данный notebook содержит сравнение базовых моделей классификации и ансамблевых моделей. Были изучены такие ансамбли как случайный лес, бэггинг, классификатор голосования, стэкинг и бустинг.
Дана оценка качества работы модеелй на основе метрики взвешенных F1-score.

# Где использовать?

Модель оценки качества вина может использоваться в бизнесе по приозводству вин, в лаборатории, а также непосредственно потребитеелм при выборе вина.
